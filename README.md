# Blog

Бэкенд версия с использованием [Laravel](https://github.com/laravel/laravel) version 5.8.0.

## Среда
- php7
- composer

## Установка

 - установить из репозитория `git clone https://github.com/laravel/laravel.git`
 - composer update
 - php artisan serve
